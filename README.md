# AI Thumbnail Generator

An AI-powered thumbnail generation platform built using React.js, Node.js, Express.js, MongoDB Atlas, JWT Authentication, Groq AI, and Pollinations AI.

## Features

* User Authentication (Signup/Login)
* JWT Authentication
* AI Thumbnail Generation
* Thumbnail Recreation
* Credit-Based System
* Community Feed
* Personal Thumbnail Library
* MongoDB Database Integration

## Tech Stack

### Frontend

* React.js
* Vite
* React Router

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* JWT
* Mongoose

### AI Services

* Groq AI
* Pollinations AI

## Installation

### Clone Repository

```bash
git clone https://github.com/nnagavarapujyothsna-oss/AI-Thumbnail-Generator.git
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```
